# イベント用 リアルタイムクイズアプリ

ウェブブラウザだけで参加できる、大人数向けのイベントクイズアプリです。
参加者はQRコードを読み込み、ニックネームを登録して参加。主催者の合図で全員一斉にクイズが始まります。

## 主な機能

- 問題セットを**事前に作成・保存**して再利用（選択式 / フリー回答を問題ごとに選択）
- フリー回答は**複数の正答**を登録可能。大文字小文字・全角半角・**ひらがな/カタカナ**・空白の違いは自動で吸収して判定
- QRコード or 参加コードで参加 → ニックネーム登録 → 待機 → 主催者スタートで一斉出題
- 1問ごとに各自の画面に**正誤**＋**ランキング**を表示（上位のみ / 全員表示の切替可）
- **正解＋回答速度**で加点（同点だらけを防ぐ）
- **1問の制限時間**と**全体の制限時間**を設定可能。時間が来たら残り問題に関わらず自動終了
- いつでも「このプロジェクトを終了」で終了し、**最終ランキングボード**を表示
- ランキングボードを **画像(PNG) / CSV** でダウンロード保存
- スマホのリロードやスリープからの**再接続に対応**（得点を維持）

## ローカルで動かす

```bash
cd quiz-app
npm install
npm start
```

- 管理画面: http://localhost:3000/admin.html
- 参加者画面: http://localhost:3000/

## インターネット公開（推奨：Render 無料プラン）

1. このフォルダを GitHub リポジトリにプッシュ
2. https://render.com で "New +" → "Web Service" → 該当リポジトリを選択
3. 設定:
   - Build Command: `npm install`
   - Start Command: `npm start`
4. デプロイ後の URL（例 `https://xxxx.onrender.com`）の `/admin.html` を主催者が開く
5. 参加者は表示されたQRを読み込むだけ（回線・Wi-Fiを問わず参加可）

> Render無料プランはアクセスが無いとスリープします。イベント直前に管理画面を一度開いて起こしておいてください。

## 使い方の流れ

1. **管理画面**で問題セットを作成 → 「保存」
2. 制限時間を設定して「プロジェクトを立ち上げる」 → QRと参加コードが表示される
3. 参加者がQRを読み込み、ニックネームを入れて参加（待機状態に）
4. 主催者が「スタート」 → 全員に一斉出題
5. 「解答へ進む」で正誤＋ランキングを公開 → 「次の問題へ」で次へ
6. 「このプロジェクトを終了」または時間切れで最終ランキング表示 → 画像/CSVで保存

## 注意（セキュリティ）

管理画面 (`/admin.html`) はURLを知っていれば誰でも開けます。
社内イベント等のクローズドな利用を想定しています。一般公開する場合は、
ホスティング側のBasic認証やアクセス制限の追加を検討してください。

## データの保存場所

- 問題セット: `data/quizzes.json`（サーバー上に保存）
- 進行中のセッション・参加者・得点: サーバーのメモリ上（イベント単位の一時データ。サーバー再起動で消えます）
